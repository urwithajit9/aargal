# Aargal

**Aargal (आर्गल)** means *barrier* or *gate* — a control point that decides what passes through.

Aargal is a lightweight, self-hosted scraper detection tool designed for:
- EC2 / VPS users
- Nginx + Fail2Ban stacks
- Developers who want control without cloud lock-in

## Phase 1 Goals
- Log-based scraper detection
- Deterministic behavior scoring
- Fail2Ban integration
- Minimal dependencies
